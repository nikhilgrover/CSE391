Marty's Pac-Man Applet
last updated Wed 2001/08/12

You may have to click the game before you're able to control it.

 

Controls:

Arrow keys:       	Move
P:                	Pause
N:                	New Game
1--5:             	Select View
Mouse right-click:	Menu

 

New Features and Bug Fixes in This Version:

  • Intro animations added. Between-level animations probably next.
  • View changing bugs squashed.
  • Many bugs squashed (many more created, too, I'm sure)

 

Known Bugs:

  • pacman/ghost wrapping should move entirely off board then come back on it
