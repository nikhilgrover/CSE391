Marty's Pac-Man Applet
last updated Wed 2001/08/12

You may have to click the game before you're able to control it.
If you have the Java Plug-in v1.30_01 or higher, try the plug-in version of Pac-Man for better speed.

 

Controls:

Mouse right-click:	Option Menu (this also pauses the game)
C:                	Insert a coin
1:                	Start One-player Game
2:                	Start Two-player Game (not yet implemented)
P:                	Pause/unpause
Arrow keys:       	Move Pac-Man
F1--F5:           	Select View

 

New Features and Bug Fixes in This Version:

  • Coin/credit option
  • Java Plug-in option for speed

 

Known Bugs:

  • Demo game playback gets off-sync and starts to look wrong

 

Missing Features:

  • isometric Pac-Mania view
  • 3-D view is implemented, but it won't work without a plugin; plus it sucks
  • animated intermissions after every 2nd level
  • different game mechanics for Mrs. and Super Pac-Man
  • 2-player mode
